# stock-trend-analysis
A Python-based toolkit for stock data analysis, featuring moving average crossovers, correlation mapping, and trend visualization.
## Features
* **Data Source**: This project utilizes the yfinance library to fetch real-time and historical market data directly from Yahoo Finance, allowing for up-to-date analysis of stock trends and performance.
* **Trend Visualization**: Plots stock price data against 50-day and 200-day moving averages.
* **Correlation Analysis**: Generates a heat map to identify relationships between different assets.
* **Automated Signals**: Implements logic to detect bullish/bearish trends based on moving average crossovers.

## Requirements
To run this project, you will need the following Python libraries installed:
- `yfinance`
- `pandas`
- `matplotlib`
- `seaborn`

Install them using:
`pip install yfinance pandas matplotlib seaborn`
